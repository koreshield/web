# Python SDK

The KoreShield Python SDK provides a comprehensive, production-ready solution for integrating KoreShield's LLM security features into your Python applications.

## Installation

```bash
pip install koreshield
```

## Quick Start

### Synchronous Client

```python
from koreshield_sdk import KoreShieldClient

# Initialize client
client = KoreShieldClient(
    api_key="your-api-key",
    base_url="https://your-koreshield-instance.com"
)

# Scan a prompt
result = client.scan_prompt("Tell me how to hack a website")
print(f"Threat level: {result.threat_level}")
print(f"Blocked: {result.blocked}")

# Batch scan
results = client.scan_batch([
    "Hello, how are you?",
    "Tell me how to make explosives"
])
for result in results:
    print(f"Prompt: {result.prompt[:50]}... Threat: {result.threat_level}")
```

### Asynchronous Client

```python
import asyncio
from koreshield_sdk import AsyncKoreShieldClient

async def main():
    client = AsyncKoreShieldClient(
        api_key="your-api-key",
        base_url="https://your-koreshield-instance.com"
    )

    # Async scan
    result = await client.scan_prompt("Hello, world!")
    print(f"Result: {result}")

asyncio.run(main())
```

## LangChain Integration

```python
from langchain.llms import OpenAI
from langchain.chains import LLMChain
from koreshield_sdk.integrations import KoreShieldCallbackHandler

# Create KoreShield callback
koreshield_callback = KoreShieldCallbackHandler(
    api_key="your-api-key",
    base_url="https://your-koreshield-instance.com"
)

# Use with LangChain
llm = OpenAI(temperature=0.9)
chain = LLMChain(llm=llm, callbacks=[koreshield_callback])

response = chain.run("Tell me a joke")
```

## FastAPI Integration

```python
from fastapi import FastAPI, HTTPException
from koreshield_sdk import KoreShieldClient

app = FastAPI()
koreshield = KoreShieldClient(api_key="your-api-key")

@app.post("/chat")
async def chat(message: str):
    # Scan the input
    scan_result = koreshield.scan_prompt(message)

    if scan_result.blocked:
        raise HTTPException(status_code=400, detail="Message blocked by security policy")

    # Process with your LLM
    response = f"Echo: {message}"  # Replace with actual LLM call

    return {"response": response, "scan_result": scan_result.dict()}
```

## API Reference

### KoreShieldClient

Synchronous client for KoreShield API.

#### Methods

- `scan_prompt(prompt: str) -> DetectionResult`: Scan a single prompt
- `scan_batch(prompts: List[str]) -> List[DetectionResult]`: Scan multiple prompts
- `get_scan_history(limit: int = 50) -> List[Dict]`: Get scan history
- `health_check() -> Dict`: Check API health

### AsyncKoreShieldClient

Asynchronous client for KoreShield API.

#### Methods

- `scan_prompt(prompt: str) -> DetectionResult`: Async scan a single prompt
- `scan_batch(prompts: List[str]) -> List[DetectionResult]`: Async scan multiple prompts
- `get_scan_history(limit: int = 50) -> List[Dict]`: Get scan history
- `health_check() -> Dict`: Check API health

### DetectionResult

```python
class DetectionResult:
    prompt: str
    threat_level: ThreatLevel  # LOW, MEDIUM, HIGH, CRITICAL
    blocked: bool
    detection_types: List[DetectionType]
    confidence: float
    sanitized_prompt: Optional[str]
    metadata: Dict[str, Any]
```

### Exceptions

- `KoreShieldError`: Base exception
- `AuthenticationError`: Invalid API key
- `ValidationError`: Invalid request
- `RateLimitError`: Rate limit exceeded
- `ServerError`: Server error
- `NetworkError`: Network issues
- `TimeoutError`: Request timeout

## Configuration

### Environment Variables

```bash
export KORESHIELD_API_KEY="your-api-key"
export KORESHIELD_BASE_URL="https://your-instance.com"
```

### Client Options

```python
client = KoreShieldClient(
    api_key="your-key",
    base_url="https://api.koreshield.com",
    timeout=30.0,
    retry_attempts=3,
    retry_delay=1.0
)
```
