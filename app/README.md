# KoreShield Website

The official website for KoreShield, an open-source security platform for protecting enterprise LLM integrations from prompt injection attacks.

## About

This website is built with:
- **Framework**: TanStack Start (React + TypeScript)
- **Styling**: Tailwind CSS
- **Database**: Drizzle ORM with SQLite
- **Authentication**: Better Auth
- **Deployment**: Vercel/Netlify

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run linting
npm run lint
```

## Project Structure

- `src/` - Main application code
- `content/` - Documentation and blog content
- `public/` - Static assets
- `drizzle/` - Database migrations
- `src/db/` - Database schema and connections
