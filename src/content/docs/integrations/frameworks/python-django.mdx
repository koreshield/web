# Django Integration Guide

Koreshield can protect **Django** applications using middleware to intercept and inspect requests before they reach your views.

## Installation

```bash
pip install koreshield django
```

## Middleware Setup

Create a file named `middleware.py` in one of your apps (e.g., `yourapp/middleware.py`):

```python
import json
from django.http import JsonResponse
from django.conf import settings
from koreshield_sdk import KoreShieldClient

class KoreShieldMiddleware:
    def __init__(self, get_response):
        self.get_response = get_response
        self.client = KoreShieldClient(
            api_key=getattr(settings, "KORESHIELD_API_KEY", ""),
            base_url=getattr(settings, "KORESHIELD_URL", "https://api.koreshield.com")
        )
        self.protected_paths = getattr(settings, "KORESHIELD_PROTECTED_PATHS", [])

    def __call__(self, request):
        if request.path in self.protected_paths and request.method == "POST":
            try:
                body = json.loads(request.body)
                prompt = body.get("message") or body.get("prompt")

                if prompt:
                    result = self.client.scan_prompt(prompt)

                    if not result.is_safe:
                        return JsonResponse({
                            "error": "Blocked by KoreShield",
                            "threat_level": result.threat_level
                        }, status=403)
            except Exception:
                pass

        return self.get_response(request)
```

## Configuration

In your `settings.py`:

```python
MIDDLEWARE = [
    # ...
    'yourapp.middleware.KoreShieldMiddleware',
]

KORESHIELD_API_KEY = "ks_..."
KORESHIELD_URL = "https://api.koreshield.com"
KORESHIELD_PROTECTED_PATHS = ["/api/v1/chat", "/api/v1/generate"]
```

## Notes
- For high-performance async Django (Generic Async Views), use `__acall__` and `AsyncKoreShieldClient` instead.
- Ensure your `KORESHIELD_API_KEY` is set via environment variable in production, not hardcoded.
