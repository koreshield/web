## Prerequisites

- Docker (recommended) or Python 3.11+
- An API key for at least one supported provider (OpenAI, Anthropic, Gemini, Azure OpenAI, or DeepSeek)
- A KoreShield API key (generated after signup)

## How It Works

KoreShield is a transparent proxy. Your app sends requests to KoreShield instead of directly to the LLM provider. KoreShield:

1. Scans the prompt (and RAG context if present) for injection attempts and policy violations
2. Applies your configured rules and thresholds
3. Forwards clean requests to the provider
4. Returns the provider response — or a blocked response if the request violated policy

The proxy is fully stateless per request. No prompt data is stored unless you explicitly enable audit logging.
