# Rate Limiting

KoreShield enforces rate limits to protect your environment and downstream providers.

## Hosted Customers

Rate limits are configured in the Dashboard. Contact support for higher limits or custom plans.

## Self-hosted Customers

Rate limits are configured in your deployment settings and can be enforced per tenant or per API key.
