# Installation

## Prerequisites

- Python 3.10+
- Redis (recommended for production)
- At least one LLM provider API key

## Local Server Install

```bash
git clone https://github.com/koreshield/koreshield.git
cd koreshield/koreshield
python -m venv .venv
source .venv/bin/activate
pip install -e .
cp config/config.example.yaml config/config.yaml
```

Set required auth env:

```bash
export JWT_SECRET='replace-with-32+char-secret'
export JWT_ISSUER='koreshield-auth'
export JWT_AUDIENCE='koreshield-api'
```

Set at least one provider key:

```bash
export DEEPSEEK_API_KEY='...'
```

Start server:

```bash
koreshield start --config config/config.yaml
```

## Docker Compose (Repository)

From `koreshield/`:

```bash
docker compose up --build
```

## Verify

```bash
curl http://localhost:8000/health
curl http://localhost:8000/openapi.json | head
```
