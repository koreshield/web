# Features

KoreShield provides a comprehensive set of security capabilities for protecting LLM integrations.

## Core Security

- **[Attack Detection](/docs/features/attack-detection)** — Multi-layered prompt injection detection with keyword rules, pattern analysis, custom DSL, and ML heuristics
- **[RAG Defense Engine](/docs/features/rag-defense)** — Scan retrieved documents for indirect prompt injection before they reach your LLM
- **[Security Overview](/docs/features/security)** — How sanitization, detection, and policy enforcement work together

## Platform Features

- **[Multi-Tenancy](/docs/features/multi-tenancy)** — Tenant isolation with per-tenant API keys, quotas, config, and audit logging
- **[RBAC](/docs/features/rbac)** — Role-based access control with 5 built-in roles and granular permissions
- **[Monitoring & Alerting](/docs/features/monitoring)** — Prometheus metrics, rule-based alerting, and multi-channel notifications (Email, Slack, Teams, PagerDuty)
